Добавление новых объектов в редактор карт:

Файле obj_info.txt 
В нём написаны ID картинкок и путь к ним.
(максимальный id = 999).
Для каждой картинки желательно иметь маску с постфиксом _mask

Файл map1_info.txt
В нём описаны все объекты на карте
ID картинки из obj_info.txt, координаты, повёрнут ли объект

Управление редактором карт:

Режим добавления новых объектов [~]
Колёсико - выбор объектов
ЛКМ - поворот объекта
ПКМ - выставление объекта

Режим перетаскивания объекта [TAB]
Зажать ЛКМ - перетаскивание
ПКМ - создать клон перетаскиваемого объекта
Колёсико - повернуть объект перетаскиваемый объект

Режим удаления объектов [1]
ЛКМ - удалить объект


Управление персонажем:

ПКМ - бежать в точку
Esc - Выход
